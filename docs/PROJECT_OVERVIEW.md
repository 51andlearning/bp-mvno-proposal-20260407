# Project Overview

## What This Site Is

A standalone proposal website presenting the **BP MVNO PWA and USSD Platform** build and ongoing support offer, prepared by **MVNE (Pty) Ltd** (a DSG company).

The site is a digital, web-rendered version of the formal proposal document, designed to be shared with the BP client as a professional, branded URL — replacing or supplementing the traditional DOCX/PDF delivery.

## Target Audience

- BP procurement and technical decision-makers
- AFGRI stakeholders involved in the MVNO platform approval process

## Deployment Environment

- **Platform:** Vercel (serverless, edge-optimised)
- **Framework:** Next.js 16+ (App Router)
- **Styling:** Tailwind CSS v4 + shadcn UI
- **Language:** TypeScript
